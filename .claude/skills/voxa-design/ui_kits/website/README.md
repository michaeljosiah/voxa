- `Hero.jsx` — headline, install command, VoiceOrb + CodeBlock.
- `Features.jsx` — pipeline diagram band + 3-up feature grid.
- `Footer.jsx` — minimal footer.

Rules: one signal-gradient moment per page (hero orb/waveform); body copy ≤ 56ch; CTAs sentence case.
